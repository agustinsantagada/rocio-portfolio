# Rocio Abad — Portfolio

Next.js 14 portfolio, deployable on Vercel (free plan).

## Setup local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subir a GitHub
2. Conectar en vercel.com → "New Project" → importar repo
3. Deploy automático

## Formulario de contacto

En `app/contact/page.tsx` reemplazar `YOUR_FORM_ID` con el ID de formspree.io

## Agregar proyectos

Editar `app/data/projects.ts` y agregar imágenes a `public/images/`
